# 数值分析迭代求根可视化系统

一个交互式的数值分析可视化工具，用于展示和分析迭代求根算法的收敛过程。

## 功能特点

### 支持的算法
- **二分法 (Bisection)** - 通过不断二分区间来逼近根
- **牛顿法 (Newton)** - 使用切线逼近根，收敛速度快
- **埃特肯法 (Aitken)** - 加速线性收敛序列的收敛速度
- **弦截法 (Secant)** - 使用割线逼近根，不需要导数

### 可视化功能
- **函数曲线绘制** - 使用 D3.js 绘制函数曲线和迭代点
- **切线/割线展示** - 动态显示牛顿法的切线和弦截法的割线
- **收敛速度分析** - 对数坐标显示误差下降曲线
- **收敛盆分析** - 彩色热力图显示不同初值的收敛情况
- **详细数据表格** - 展示每次迭代的详细数据

### 交互式动画控制
- 播放/暂停动画
- 单步前进/后退
- 速度调节（0.1x - 3.0x）
- 进度条跳转

### 函数输入
- 5个预设经典测试函数
- 自定义函数输入（支持数学表达式解析）
- 参数调整（初值、容许误差、最大迭代次数）

### 数据导出
- CSV 格式导出（包含完整迭代历史）
- JSON 格式导出（包含元数据和结果）

### 双主题设计
- **现代数字艺术风格** - 浅色渐变背景，柔和色彩，极简主义
- **科学仪器风格** - 深色主题，霓虹色数据线，示波器风格

## 技术栈

- **React 18** - 前端框架
- **Vite** - 构建工具
- **D3.js v7** - 数据可视化
- **math.js** - 数学表达式解析
- **CSS Modules** - 样式管理

## 使用说明

### 1. 选择算法
在左侧面板选择四种算法之一：二分法、牛顿法、埃特肯法或弦截法。

### 2. 设置函数
- 从预设函数中选择，或
- 输入自定义数学表达式（支持 +, -, *, /, ^, sqrt(), sin(), cos(), exp(), log() 等）

### 3. 调整参数
- **初值/区间起点**: 算法的起始点或区间左端点
- **第二个初值/区间终点**: 弦截法的第二个初值或二分法的区间右端点
- **容许误差**: 收敛判断的误差阈值
- **最大迭代次数**: 防止无限循环

### 4. 查看结果
- 自动计算并显示迭代过程
- 使用动画控制器逐步观察收敛过程
- 查看收敛速度分析和详细数据表格

### 5. 切换主题
点击右上角的主题切换按钮，在现代风格和科学仪器风格之间切换。

### 6. 导出数据
点击"导出 CSV"或"导出 JSON"按钮保存迭代数据。

## 算法说明

### 二分法
- **收敛阶**: 线性收敛
- **特点**: 稳定可靠，但收敛较慢
- **要求**: 需要函数在区间端点异号

### 牛顿法
- **收敛阶**: 二次收敛
- **特点**: 收敛速度快
- **要求**: 需要函数的导数

### 埃特肯法
- **收敛阶**: 二次收敛（加速后）
- **特点**: 可加速线性收敛序列
- **要求**: 基于不动点迭代

### 弦截法
- **收敛阶**: 超线性收敛（约 1.618）
- **特点**: 不需要导数，收敛速度介于二分法和牛顿法之间
- **要求**: 需要两个初值

## 预设函数

1. **x² - 2** - 求 √2
2. **sin(x) - x/2** - 超越方程
3. **x³ - x - 2** - 三次方程
4. **e^x - 3** - 指数方程
5. **x⁴ - 2x³ - 4x² + 4x + 4** - 高次多项式
